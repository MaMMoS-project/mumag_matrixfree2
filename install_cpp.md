# Installation Guide for Micromagnetics C++ (VexCL + AMGCL)

This guide covers the installation of dependencies for Ubuntu 24.04 (Noble Numbat) with a GPU (OpenCL/CUDA).

## 1. System Packages
Install the core build tools, Eigen for linear algebra, Boost for AMGCL, and OpenCL headers/drivers.

```bash
sudo apt update
sudo apt install -y \
    build-essential \
    cmake \
    libboost-all-dev \
    libeigen3-dev \
    libopencl-clang-dev \
    opencl-headers \
    ocl-icd-opencl-dev \
    zlib1g-dev \
    git
```

## 2. Hardware Drivers
Ensure you have the appropriate OpenCL drivers for your GPU:
- **NVIDIA:** `sudo apt install nvidia-cuda-toolkit`
- **AMD:** `sudo apt install mesa-opencl-icd`
- **Intel:** `sudo apt install intel-opencl-icd`

Verify with `clinfo`.

## 3. Header-Only Dependencies
We keep these in a `deps/` folder within the project.

```bash
mkdir -p deps
cd deps

# VexCL (GPU vector/sparse matrix library)
git clone https://github.com/ddemidov/vexcl.git

# AMGCL (Algebraic Multigrid library)
git clone https://github.com/ddemidov/amgcl.git
```

## 4. `cnpy` (NPZ File Support)
This is used to read the mesh files generated by the Python scripts.

```bash
cd deps
git clone https://github.com/rogersce/cnpy.git
mkdir -p cnpy/build
cd cnpy/build
cmake ..
make
sudo make install
```

## 5. Building the Project
Once dependencies are installed, build the micromagnetics C++ code:

```bash
mkdir build
cd build
cmake ..
make
```

## 6. Testing
Run the Poisson convergence test:
```bash
# Generate a test mesh first using Python
python3 ../src/mesh.py --geom box --extent 20,20,20 --h 2.0 --out-name cube_20nm
python3 ../src/add_shell.py --in cube_20nm.npz --layers 4 --out-npz cube_with_shell.npz

# Run C++ solver
./test_poisson_convergence ../cube_with_shell.npz
```
